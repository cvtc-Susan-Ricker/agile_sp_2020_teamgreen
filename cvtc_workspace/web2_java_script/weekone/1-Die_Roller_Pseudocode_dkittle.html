Die Roller Pseudocode

Name: David Kittle

Problem Statement

Write a program that will simulate the roll of an input number of dice. The solution should interact naratively with the user by first name, and randomly generated a number to simulate the roll of each single die requested. The output should show the random result for each separate die and then calculate the sum of the requested number of dice. Random numbers generated for each die must result in single digit integers between 1 and 6 inclusive.

1. Nouns and Verbs

Identify the nouns and verbs in the problem statement.

PROBLEM STATEMENT:
Write a program that will simulate the roll of an input number of dice. The solution should
interact naratively with the user by first name, and randomly generated a number to simulate
the roll of each single die requested. The output should show the random result for each
separate die and then calculate the sum of the requested number of dice. Random numbers
generated for each die must result in single digit integers between 1 and 6 inclusive.

Nouns: first name, number of dice, randomly-generated die rolls, Sum of dice

Verbs: input number of dice, simulate roll of die, generate number between 1-6, show each die and sum of dice

2. Defining Diagram

Convert the nouns and verbs into variable names and processes and create a defining diagram.

INPUTS:
first name
number of dice

PROCESSING:
  prompt use for first name
  prompt user for number of die(dice)
  for each die
    randomly generate a number between 1 and 6
    display each die number
  display sum of dice

OUTPUTS:
  each die number
  sum of dice

3. Solution Algorithm

Enter the Solution Algorithm below. 

  SimulateRollingDice
    initialize strFirstName to ""
    initialize intNumberDie to 0
    initialize intDieValue to 0
    initialize intSumOfDice to 0

    prompt for strFirstName with "Please enter your first name."
    validate strFirstName
    prompt for intNumberDie with "How many dice would you like to roll?"
    validate intNumberDie

    display "Rolling " + intNumberDie + " "

    set strDieWord = "dice"
    IF intNumberDie == 1 THEN
      strDieWord = "die"
    ENDIF

    display strDieWord + "...." + NewLine
    display "Hey " + strFirstName + " you rolled a "

    FOR var intCounter = 1 to intNumberDie step 1

      intDieValue = Random(1-6)

      display pop-up "Dice #" + intCounter + " is a " + intDieValue

      IF intCounter != intNumberDie OR intNumberDie == 1 THEN

        IF intNumberDie <= 2 THEN
           display intDieValue + " "
        ELSE
           display intDieValue + ", "
        ENDIF

      ELSE
        display "and " + intDieValue + " "
      ENDIF

      intSumOfDice = intSumOfDice + intDieValue

    ENDFOR

    display " for a grand total of " + intSumOfDice + NewLine + NewLine

  END
